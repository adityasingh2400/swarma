SWARMA DEV PREVIEW FLAGS
========================

Add ?preview=<flag> to the frontend URL to jump directly to any page
with mock data, bypassing the normal pipeline flow.

Base URL: http://localhost:5174

FLAG                  URL                                        DESCRIPTION
----                  ---                                        -----------
intake                ?preview=intake                            Upload/drop zone + phone QR panel
research              ?preview=research                          Per-item research universe with orbital route panels (carousel)
decisions             ?preview=decisions                         Route decision cards with carousel (2+ items)
market-sweep          ?preview=market-sweep                      Horizontal card rails: resale, trade-in, repair listings
best-route            ?preview=best-route                        Route comparison ladder (ranked alternatives)
quote-sweep           ?preview=quote-sweep                       Trade-in quote cards with confidence bars
repair-sweep          ?preview=repair-sweep                      Repair analysis: defects, parts, payoff comparison
bundle-merge          ?preview=bundle-merge                      Bundle flow: individual items -> combined value
asset-studio          ?preview=asset-studio                      Image frame grid with raw/enhanced slider
posting-workspace     ?preview=posting-workspace                 Listing creation: decision cards + browser mock grid
unified-inbox         ?preview=unified-inbox                     Message threads with buyer conversations
condition-fusion      ?preview=condition-fusion                  Video condition analysis with item extraction
multi-post            ?preview=multi-post                        Multi-platform posting status tracker

QUICK START
-----------
1. cd frontend/mac && npm install && npm run dev
2. Open http://localhost:5174/?preview=decisions
3. Click page links in the nav bar to switch between previews

NOTES
-----
- All previews use mock data (4 items, randomized prices/bids)
- The processing page (AgentTheater, MissionControl, SwarmGrid) has no preview flag
- Images come from picsum.photos (requires internet)
- Carousel activates on research and decisions pages when 2+ items are present
- Glass/shine effects visible on market-sweep, decisions, and intake cards
